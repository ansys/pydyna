Modules
=======
PyDYNA consists of four modules, ``ansys.dyna.core.pre``, ``ansys.dyna.core.solver``,
``ansys.dyna.core.run``, and ``ansys.dyna.core.keywords``.
learn about the different modules in PyDYNA.
