ansys.dyna.core.lib.keyword_collection.KeywordCollection#

class ansys.dyna.core.lib.keyword_collection.KeywordCollection(keywords: Iterator[ansys.dyna.core.lib.keyword_base.KeywordBase] | List[ansys.dyna.core.lib.keyword_base.KeywordBase])#

A lazy, chainable wrapper around keyword iterables.

This class provides a fluent interface for filtering and accessing keywords without materializing intermediate results until explicitly requested.

Examples

>>> collection = KeywordCollection(deck.keywords)
>>> shells = collection.by_subtype("SHELL")
>>> for kwd in shells:
...     print(kwd.secid)
>>> first_shell = shells.first()
>>> all_shells = shells.to_list()
