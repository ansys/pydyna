ansys.dyna.core.keywords.keyword_classes.manual.section_tshell_version_0_11_0.SectionTShellLegacy#

class ansys.dyna.core.keywords.keyword_classes.manual.section_tshell_version_0_11_0.SectionTShellLegacy(**kwargs)#

Bases: ansys.dyna.core.lib.keyword_base.KeywordBase

Legacy DYNA SECTION_TSHELL keyword (pydyna 0.11.0 API).

keyword = 'SECTION'#
subkeyword = 'TSHELL'#
option_specs#

Gets all option specs for this keyword.

property secid: int | None#

Get or set the Section ID. SECID is referenced on the *PART card and must be unique.

property elform: int#

Get or set the Element formulation: EQ.1: one point reduced integration (default), EQ.2: selective reduced 2x2 in plane integration. EQ.3: assumed strain 2x2 in plane integration. EQ.5: assumed strain reduced integration. EQ.6: assumed strain reduced integration with shell materials EQ.7: assumed strain 2×2 in plane integration

property shrf: float#

Get or set the Shear factor (default =1.0). A value of 5/6 is recommended.

property nip: int#

Get or set the Number of through shell thickness integration points (default = 2).

property propt: float#

Get or set the Printout option: EQ.1.0: average resultants and fiber lengths (default), EQ.2.0: resultants at plan points and fiber lengths, EQ.3.0: resultants, stresses at all points, fiber lengths.

property qr: int#

Get or set the Quadrature rule: LT.0: absolute value is specified rule number, EQ.0: Gauss (up to five points are permitted), EQ.1: trapezoidal, not recommended for accuracy reasons.

property icomp: int#

Get or set the Flag for layered composite material mode: EQ.0: Flag turned off (default), EQ.1: a material angle is defined for each through thickness integration point . For each layer one integration point is used.

property tshear: int#

Get or set the Flag for transverse shear strain or stress distribution (see Remarks 4 and 5): EQ.0.0: Parabolic, EQ.1.0: Constant through thickness.

property bi: ansys.dyna.core.lib.series_card.SeriesCard#

Dynamic array of beta-i: material angle at ith-integration point..

property title: str | None#

Get or set the Additional title line
