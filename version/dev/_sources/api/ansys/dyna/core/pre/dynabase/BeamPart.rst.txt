





:class:`BeamPart`
=================


.. py:class:: ansys.dyna.core.pre.dynabase.BeamPart(pid)

   Bases: :py:obj:`Part`


   
   Defines a beam part.

   A beam part definition consists of the combined material information,
   section properties, hourglass type, thermal properties, and a flag
   for part adaptivity.
