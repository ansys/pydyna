   Evaluate an expression and return the result with appropriate type.


   :Parameters:

       **expression** : :class:`python:str`
           Expression string to evaluate (e.g., "a+b*2").

       **param_type** : :class:`python:str`

   :Returns:

       :class:`python:int` or :class:`python:float`

   :Raises:

       :obj:`ValueError`
