ansys.dyna.core.lib.table_card.TableCard
========================================

.. py:class:: ansys.dyna.core.lib.table_card.TableCard(fields: List[ansys.dyna.core.lib.card.Field], length_func, active_func: Optional[Callable] = None, name: str = None, format: ansys.dyna.core.lib.format_type.format_type = format_type.default, **kwargs)

   Bases: :py:obj:`ansys.dyna.core.lib.card.Card`


   
   A Card represents a single line of data in a keyword.

   This implementation separates schema (shared) from values (per-instance)
   for memory efficiency and faster loading of keywords with many cards.
